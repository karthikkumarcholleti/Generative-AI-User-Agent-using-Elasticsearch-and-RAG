# Research Paper Sections - Quick Reference

## Your 4 Sections

1. **LITERATURE_REVIEW.md** - Review existing work (1500-2000 words)
2. **METHODOLOGY.md** - Describe your system (2000-2500 words)
3. **RESULTS.md** - Present evaluation results (1500-2000 words)
4. **DISCUSSION.md** - Interpret and discuss (1500-2000 words)

## How to Use These Templates

1. **Read each template** to understand structure
2. **Fill in content** based on your system
3. **Use examples** as guides for writing style
4. **Add citations** as you write
5. **Include figures/tables** where indicated
6. **Review word counts** for each section

## Before Writing

1. ✅ Decide on target journal/conference
2. ✅ Check journal format requirements
3. ✅ Review journal's author guidelines
4. ✅ Take screenshots using queries from RESEARCH_PAPER_TEST_QUERIES.md
5. ✅ Organize figures with clear naming

## Writing Order Recommendation

1. **Start with Methodology** (you know this best)
2. **Then Results** (what you've tested)
3. **Then Literature Review** (research background)
4. **Finally Discussion** (interpret everything)

## Questions to Discuss with Professor

1. Target journal/conference?
2. Paper length requirements?
3. Citation format (IEEE, APA, etc.)?
4. Figure/table limits?
5. Co-author contributions statement?
6. Timeline/deadline?

## Key Reminders

- ✅ Write in third person
- ✅ Use past tense for methods/results
- ✅ Use present tense for general statements
- ✅ Be specific and precise
- ✅ Cite recent work (2020-2024)
- ✅ Acknowledge limitations honestly
- ✅ Highlight your contributions clearly
